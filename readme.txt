== Description ==

Loan Calculator is a special application that has been deliberately created for all types of users. a housewife, a student, a businessperson, an investor, a banker, a certified public accountant, a financial planner, etc.

Shortcode - **[loan_calculator]**

<h3>Features</h3>

<ul>
<li>Mobile friendly UI</li>
<li>Customize calculator body background color</li>
<li>Change calculator heading color with font size</li>
<li>Change calculator heading color with font size</li>
<li>Change calculator loan detail title and calculator texts color</li>
<li>Customize slider active and progress color</li>
<li>Customize slider thumb background and border color</li>
<li>Change result title color</li>
<li>Enable/ Disable total payment break-up and yearly payment breakdown chart</li>
<li>Customize payment break-up chart with chart type and color</li>
<li>Customize yearly payment breakdown chart with principal and interest paid color</li>
<li>Customize yearly payment breakdown chart with principal and interest paid color</li>
<li>Change calculator title text</li>
<li>Customize result title text setting</li>
<li>Customize payment break-up and yearly payment breakdown title text setting</li>
<li>Set min and max loan amount with default loan amount</li>
<li>Set min and max loan loan terms with default loan year</li>
<li>Set min and max loan loan interest with default loan interest</li>
<li>Loan Calculator</li>
<li>Mortgage Calculator</li>
<li>Add Investment</li>
</ul>
